The Purpose of the Program

This is a POS system for Best Buy Retail Store. The purpose of this program is to allow cashiers to manage customer purchases. They can: 

* Add items to the shopping cart
* Remove items 
* View the cart 
* Calculate the total bill (inclusive of taxes and discounts where applicable)
* Accept payments
* Generate a Receipt. It also tracks inventory. 



How to Run the Program

To run the program, you'll need to:

1. Install Python (the file is a .py file that requires Python to open)
2. Ensure Native.Coders-POS-ITT103-SP2026.py is saved to your device 
3. Open (run) it from the folder where you saved it 

Once the screen is up, you'll see the menu populated. Follow the prompts to interact with the system and make your selection. 



Required Modifications

A Graphic User Interface(GUI) for better usability.



Assumptions

Assuming that the system will fulfil the needs of all users who interact with it. It can only be used by one user at a time.



Limitations

* The system doesn't have authentication access; anyone can save the file and access it
* Changes would have to be made to every computer on which the program is written if any modifications are needed
* This is a standalone software that is not connected to a network and can only run on one computer
* It does not save the information after you exit; each time you open the file, the inventory resets
